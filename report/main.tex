%%%%%%%%% ABSTRACT
\begin{abstract}
精细抠图要在头发、毛发与半透明边界上估计连续透明度 $\alpha$，是背景替换、虚化与视频合成的共同前提。
现有路线存在割裂：传统抠图依赖人工 trimap 或专门训练，而以 SAM 为代表的视觉基础模型擅长定位、
却只输出粗糙的二值掩码。我们提出 PromptMatte，一个\emph{零训练}的精细抠图与换背景框架，
将开放词汇定位、精细 $\alpha$ 估计、视频时序稳定与背景编辑统一为单一推理流水线，全程不更新任何模型权重。
其核心是一条可跨域复用的\emph{一致性先验}：好的 $\alpha$ 在变换下应当保持一致。
在图像端，该先验落地为翻转等变的测试时增强与边缘投影（PromptMatte-TTA-GF），
并辅以以检测框为空间先验的 Box-Support Prior；在视频端，同一先验沿时间轴推广为双向时序平滑，
并复用图像分支的首帧结果作为种子。在 MicroMat-3K 的 valid706 子集上，
本方法相对 ZIM 官方 bbox 基线在 SAD、MSE、MAE 与 Boundary~SAD 上分别下降
$8.5\%$、$23.1\%$、$8.8\%$ 与 $13.4\%$；在 20 段视频上，双向平滑是唯一在几乎无精度损失下
提升时序一致性的策略。全部增益均不依赖训练，方法可解释、易复现、模块可随基础模型升级而替换。
\end{abstract}

\label{fig:teaser}

抠图（image matting）求解的是欠定的合成方程
\begin{equation}
C=\alpha F+(1-\alpha)B,\qquad \alpha\in[0,1],
\label{eq:matting}
\end{equation}
每个像素仅观测到混合颜色 $C$，却需同时反解前景 $F$、背景 $B$ 与透明度 $\alpha$。
任务的难点不在“框出目标”，而在头发、毛发、半透明与细结构处对连续 $\alpha$ 的估计——
这正是背景替换、虚化与海报合成等编辑能否干净的关键。

两条主流路线各有掣肘。传统抠图借助人工 trimap 将图像划分为确定前景、确定背景与不确定边界带，
把求解收缩到窄带内，但 trimap 标注昂贵、难以规模化；近年的可提示分割模型~\cite{sam,sam2}
具备强大的零样本定位能力，却只输出接近二值的硬掩码，在发丝处只能硬切。
一个自然的问题是：能否在\emph{不训练}的前提下，把“准确定位”与“精细 $\alpha$”衔接成一个可用系统？

我们给出的答案是 PromptMatte。它不重新训练任何大模型，而是复用现成的视觉基础模型，
把全部设计放在推理阶段的先验注入上。贯穿系统的是一条简单的\emph{一致性先验}：
正确的 $\alpha$ 在变换下应当保持一致。该先验在图像上体现为翻转等变——同一图像翻转后预测应与原图一致；
在视频上自然推广为时序一致——相邻帧的 $\alpha$ 应当连续。两者共享同一形式，使图像分支与视频分支
并非彼此独立，而是同一方法在空间与时间上的两次实例化。

本文的贡献为：
\begin{itemize}
\item 提出零训练的精细抠图与换背景框架 PromptMatte，将 SAM2 定位、ZIM 抠图、
视频时序稳定与背景编辑统一为单一推理流水线（\secref{sec:method}）。
\item 在上游定位端，提出 Prompt Ensemble 与 IoU 引导重排序，以多策略候选与综合打分
显著提升 bbox 提示下的掩码稳定性（\secref{sec:upstream}）。
\item 在 ZIM 输出之上以\emph{先验注入}方式提出 PromptMatte-TTA-GF 与 Box-Support Prior，
分别约束边界对齐与前景空间范围，二者互补且不需任何训练（\secref{sec:matting}）。
\item 将一致性先验沿时间推广为双向时序平滑，并复用图像首帧结果，使视频分支自然接入主线（\secref{sec:video}）。
\item 在 valid706 与 20 段视频上系统验证，本方法在多项指标上一致优于强基线，
并给出可解释的消融归因（\secref{sec:exp}）。
\end{itemize}

\section{相关工作}
\label{sec:related}

\noindent\textbf{可提示分割基础模型。}
SAM~\cite{sam} 确立了可提示分割范式，SAM2~\cite{sam2} 将其扩展到图像与视频、
并以 Hiera 主干~\cite{hiera}（一种层次化视觉 Transformer~\cite{vit}）显著提速；
Grounded-SAM~\cite{groundedsam} 引入文本编码器实现开放词汇定位。这类模型回答“目标在哪里”，
对每个提示给出多候选掩码及 IoU 置信度，但输出接近二值，并不直接提供软 $\alpha$。
我们以 SAM2 作为上游定位器，仅取其 bbox 提示能力。

\noindent\textbf{图像抠图。}
针对 \equref{eq:matting} 的欠定性，传统方法以 trimap 约束求解空间，
代表性工作包括闭式解~\cite{closedform} 与基于深度学习的 Deep Image Matting~\cite{dim}、IndexNet~\cite{indexnet}；
免 trimap 路线如 MODNet~\cite{modnet} 面向人像实时抠图，ViTMatte~\cite{vitmatte} 将预训练 ViT 引入抠图。
在基础模型方向，Matting Anything~\cite{mam} 尝试在 SAM 之上做抠图；ZIM~\cite{zim} 面向零样本抠图，
保留点/框提示交互，并以层次化像素解码器输出连续 soft matte、无需抠图标注即可训练。
我们以 ZIM 为抠图基座，用 SAM2 自动产生的框提示取代人工 trimap，并在其输出上注入先验细化。

\noindent\textbf{视频抠图与时序稳定。}
Background Matting v2~\cite{bgmv2} 与 Robust Video Matting~\cite{rvm} 面向实时高分辨率视频抠图；
MatAnyone~\cite{matanyone0} 以一致性记忆传播稳定视频，其后续 MatAnyone2~\cite{matanyone} 面向人像视频抠图，
但逐帧推理仍带来 $\alpha$ 抖动与边界闪烁，换背景时会被放大。
我们不训练额外时序模型，而以一致性先验对边界带做时序平滑。

\noindent\textbf{测试时增强与边缘感知滤波。}
测试时增强通过对输入施加变换并融合预测以降低方差；引导滤波~\cite{gf} 以引导图的边缘结构约束输出、
实现保边平滑。我们将两者组合，作为翻转等变一致性先验的具体实现。

\noindent\textbf{零训练与推理时适配。}
在算力与标注受限的场景，免训练地复用基础模型日益重要。测试时增强、测试时适配与基于先验的后处理
是三类常见手段：前两者多用于降低方差或在线适配分布，后者用于注入领域知识。
本文沿此脉络，但区别在于强调\emph{同一先验在不同模态间的复用}——图像端的翻转等变与视频端的时序一致
共享相同形式，从而以最小的设计冗余同时覆盖图像与视频两类任务，而非为每个任务各自设计一套启发式后处理。

\section{方法}
\label{sec:method}

\subsection{框架总览}
\label{sec:overview}
PromptMatte 是一条从定位到合成的单一推理流水线，分图像与视频两条分支，全程冻结所有模型（\figref{fig:pipeline}）。
图像主线为：图像与 official bbox 输入后，SAM2 给出候选掩码，经 Prompt Ensemble 与 IoU 引导重排序
择优、再以引导滤波修边，得到稳定的粗定位；ZIM 据此输出连续 $\alpha$；在该 $\alpha$ 之上依次注入
TTA-GF 与 BSP 两个先验；最终经 Composer 输出 RGBA、背景替换或虚化。视频分支不另起炉灶，
而是复用图像主线的首帧 $\alpha$ 作为种子，交 MatAnyone2 得到逐帧结果，再以双向时序平滑稳定边界。

\begin{figure*}[t]
\centering
\includegraphics[width=0.98\textwidth]{pipeline.png}
\caption{PromptMatte 整体流水线。\textbf{上：}图像主线，从 official bbox 经 SAM2 与候选重排序得到粗定位，
由 ZIM 输出连续 $\alpha$，再注入 TTA-GF 与 BSP 两个先验后合成。\textbf{下：}视频支线复用图像主线的首帧 $\alpha$
（虚线箭头），经 MatAnyone2 与双向时序平滑稳定边界。灰框为冻结的开源模块，紫框为我们以零训练方式注入的先验。}
\label{fig:pipeline}
\end{figure*}

关键在于，全部增益都不来自训练，而来自“在恰当位置注入恰当先验”：我们把领域知识——
变换一致性、目标必在框内、相邻帧连续——显式写入推理。这使方法可解释（每步增益可归因于一条先验）、
易复现（无需训练资源）、且与基础模型解耦（上游定位与抠图基座均可平滑替换）。

\subsection{上游开放词汇定位}
\label{sec:upstream}
该模块回答“目标在哪里”，为下游抠图提供稳定入口。我们选用 SAM2.1 Hiera-Base+：
其轻量主干、原生 bbox/points 提示与多候选 $+$ IoU 置信度，恰好为重排序提供可靠信号；
相较 SAM3 推理更快，相较 Grounded-SAM-2 无需额外文本编码器。

\noindent\textbf{多提示候选与 IoU 引导重排序。}
单次 bbox 推理易受局部纹理干扰而选到次优掩码。我们用多组提示策略
（\emph{bbox-only} 与 \emph{bbox $+$ 正点}）生成候选 $\{m\}$，并以综合得分择优：
\begin{equation}
S(m)=0.55\,\mathrm{IoU}(m)+0.25\,\mathrm{conn}(m)+0.20\,\mathrm{cov}(m),
\label{eq:rerank}
\end{equation}
其中 $\mathrm{IoU}$ 为模型自评的掩码质量（最可靠信号），$\mathrm{conn}$ 以连通分量数惩罚碎片化，
$\mathrm{cov}$ 约束掩码对检测框的合理覆盖。权重次序反映了三类信号的可靠性：
IoU 是模型对自身预测的直接置信，居于主导；连通性与框覆盖是无需 GT 的结构先验，
用于在 IoU 相近时打破并列、剔除碎片化或明显越界的候选。每个提示策略经 SAM2 给出至多三个候选，
多策略候选合并后在统一得分下择优，等价于一次轻量的候选集成——它不改变模型、不增加可学习参数，
却显著降低了单次推理“选错候选”的概率，这一离散错误恰是小目标失败的主因。
可选的多尺度推理在 $\{0.75,1.0,1.25\}\times$ 上分别预测、对齐后平均，从 bagging 的角度进一步稀释单尺度的随机偏差；
它与重排序互补，但带来约三倍开销，故仅作为可选项，未进入最终配置。

\subsection{精细抠图：先验注入的边界细化}
\label{sec:matting}
该模块回答“边界够不够精细”，是本文的方法核心。ZIM 沿用提示式架构，可拆为四步（\figref{fig:zim}）：
图像编码器把原图编码为视觉特征；提示编码器把框/点编码为提示特征并圈定大致目标区域；
解码器以掩码注意力让两类特征交互、聚焦目标、抑制背景；最后由\emph{层次化像素解码器}逐级上采样、
融合多尺度特征，输出逐像素的连续 $\alpha$ 而非二值掩码。其中末端的像素解码器是 ZIM 区别于 SAM 的关键——
前三步负责“目标在哪”，它负责“把边界做细”。我们将整个 ZIM 视为\emph{冻结的黑盒}——给框、出 $\alpha$，
不改其任何权重，所有优化都作用在它的 soft matte 输出之上。这一选择使方法与基座解耦：
若日后出现更强的零样本抠图模型，可直接替换 ZIM 而保留全部先验设计。

\begin{figure}[t]
\centering
\includegraphics[width=\linewidth]{zim_overview.png}
\caption{ZIM 原理。输入图像与提示，经编码器与解码器，由像素解码器输出连续 soft matte；
我们在该输出上注入先验，不改动其权重。图源：ZIM 官方项目~\cite{zim}。}
\label{fig:zim}
\end{figure}

\noindent\textbf{PromptMatte-TTA-GF。}
ZIM 的单次预测在边界处对视角方向与局部纹理敏感，同一目标换一朝向即产生偏差。
我们将其建模为翻转等变一致性：理想情况下 $f_\theta(F(\mathcal{I}))$ 应等于 $F(f_\theta(\mathcal{I}))$，
其中 $F$ 为水平翻转。据此对原图与翻转各预测一次，对齐后平均以抵消方向相关的随机误差，
再以原图 RGB 边缘为引导、用引导滤波将 $\alpha$ 投影到真实图像边缘：
\begin{align}
\alpha_0 &= f_\theta(\mathcal{I},b), &
\alpha_1 &= F^{-1}\!\big(f_\theta(F(\mathcal{I}),F(b))\big), \notag\\
\alpha_T &= \tfrac{1}{2}(\alpha_0+\alpha_1), &
\alpha_G &= \mathrm{GF}(\mathcal{I},\alpha_T).
\label{eq:ttagf}
\end{align}
这里的关键观察是：理想的等变模型应满足 $f_\theta(F(\mathcal{I}))=F(f_\theta(\mathcal{I}))$，
但真实网络存在等变误差，且原图与翻转两支在边界上的误差方向\emph{不相关}；
对齐平均后，这部分方向相关的随机误差相互抵消，如同集成的方差缩减。
引导滤波则在每个局部窗口内假设输出对引导图 RGB 近似线性，通过最小二乘拟合使 $\alpha$
既贴近 $\alpha_T$、又跟随原图的真实边缘结构：图像有强边处 $\alpha$ 被锐化归位，平坦处被平滑，
从而在不抹糊发丝的前提下校正边界偏移。两步皆为推理时操作，不引入任何可学习参数（\figref{fig:ttagf}）。

\begin{figure*}[t]
\centering
\includegraphics[width=0.97\textwidth]{ttagf.png}
\caption{PromptMatte-TTA-GF。原图与其水平翻转分别经冻结的 ZIM 预测，翻转支经 $F^{-1}$ 对齐后与原图支平均
得到 $\alpha_T$（翻转等变一致性）；再以原图 RGB 边缘为引导、用引导滤波将 $\alpha_T$ 投影到真实图像边缘
得到 $\alpha_G$（边缘感知投影）。全程冻结 ZIM、无可学习参数。}
\label{fig:ttagf}
\end{figure*}

\noindent\textbf{Box-Support Prior。}
TTA-GF 之后，少数样本仍在 official bbox 外残留 $\alpha$。利用“目标必在框内”这一可靠的空间先验，
我们由检测框构造软支持图 $G$（框内为 1、框外渐变为保守下限 $\eta$），对 $\alpha_G$ 做温和重加权：
\begin{equation}
\alpha_B(x)=\alpha_G(x)\,\big(1-\lambda+\lambda\,G(x)\big).
\label{eq:bsp}
\end{equation}
框内（$G\!\approx\!1$）保持不变，框外（$G\!\approx\!0$）按 $(1-\lambda)$ 压低；
$\eta$ 设定框外的保守下限以免“一刀切”，$\lambda$ 控制抑制强度。
为避免误伤贴近框边的合法前景，支持图在构造时对框做轻微外扩并对边缘做高斯模糊，
使框内外平滑过渡。BSP 因此是一项保守的稳定化：它几乎不触碰主体，只清除框外不可能为真前景的残留，
作为 TTA-GF 之上的轻量补充而不引入回退。两个先验分工互补：TTA-GF 约束边界\emph{对齐}，BSP 约束前景\emph{范围}。

\subsection{视频时序平滑}
\label{sec:video}
视频分支以 MatAnyone2 为基线，但其逐帧推理带来 $\alpha$ 抖动。我们做两点设计。
其一，\emph{首帧复用}：直接以图像主线算出的首帧 $\alpha$ 作种子，把图像端的全部细化带入视频，
而非从零开始。其二，把图像端的一致性先验沿时间轴推广——相邻帧 $\alpha$ 应当连续——
施加\emph{双向}（同时利用过去与未来帧）时序平滑，且仅作用于边界带 $\alpha\in[0.05,0.95]$，
从而保护前景核区与发丝、避免过度模糊。选择双向而非单向 EMA，是因为单向滤波只参考历史、
会引入系统性的相位滞后，使快速运动的边界“拖影”；双向对称地融合过去与未来，既稳定又不滞后。
将作用域限制在边界带，则把“稳定时序”与“保住细节”解耦：核区与背景已确定、无需平滑，
只有不确定的边界带才施加约束。该平滑复用邻帧信息而非训练时序模型，
是同一先验在时间维度上的落地，而非事后修补。

\subsection{复杂度}
\label{sec:complexity}
额外开销集中在推理且多源于可裁剪模块：SAM2.1 Hiera-Base+ 在 A10 上约 $1.16$ it/s（bbox-only）；
Prompt Ensemble 近似线性增加前向次数，多尺度带来约三倍开销，二者皆可选。
TTA 使 ZIM 前向翻倍，引导滤波与 BSP 为轻量逐像素后处理；双向平滑为一维时序滤波、与帧数线性相关。
由于全程不训练，系统无需训练资源即可复现，部署时可按精度/时延关闭可选模块，
仅保留 “ZIM\,$+$\,TTA-GF\,$+$\,BSP” 的核心链路。

\section{实验}
\label{sec:exp}

\subsection{实验设置}
\label{sec:setup}
图像端使用 MicroMat-3K 验证集中带 official bbox 的 valid706 子集（706 张图像，每张配官方提示与 GT $\alpha$）；
视频端使用 YouTubeMatte 与 VideoMatt 各 10 段、共 20 段。
图像指标为 SAD、MSE、$\mathrm{MAE}\times10^3$ 与 Boundary~SAD（GT 边界 5px 带内统计，专测边缘），均越低越好；
SAD 与 Boundary~SAD 按抠图惯例以 $/1000$ 计，故表中为个位数量级。视频指标为时序一致性 TC（报告相对提升）
与 SAD/MSE 变化。为保证公平，所有方法共享同一 official bbox、同一脚本评分，Composer 应用不计入指标。
上游为 SAM2.1 Hiera-Base+，抠图基座为 ZIM ViT-B，引导滤波半径取 1，视频双向平滑取 $\alpha=0.7$。

\noindent\textbf{指标定义。}记预测与真值分别为 $\hat\alpha$、$\alpha^*$。
$\mathrm{SAD}=\sum_x|\hat\alpha(x)-\alpha^*(x)|$ 度量整体绝对误差，MSE 与 MAE 分别为均方与平均绝对误差；
Boundary~SAD 仅在 GT 边界 5px 带内统计 SAD，专门刻画发丝等边缘区域的质量。
视频端 TC 度量相邻帧 $\alpha$ 的时序波动，我们报告其相对基线的下降比例，即时序一致性的提升。

\subsection{主结果}
\label{sec:exp-main}
\tabref{tab:main} 在统一协议下对比两阶段派生的 $\alpha$ 质量。两点结论清晰：
其一，精细抠图基座不可替代——即便上游经 ensemble 与引导滤波优化到最优（SAD $2.774$），
仍逊于 ZIM 的 bbox 基线（$2.320$），印证了“定位强不等于 $\alpha$ 精细”。
其二，本文先验在 ZIM 之上带来一致提升：相对 ZIM 基线，TTA-GF 与 BSP 叠加后
SAD、MSE、MAE、Boundary~SAD 分别下降 $8.5\%$、$23.1\%$、$8.8\%$、$13.4\%$，
其中 MSE 与边界误差降幅最大——这与“注入边界一致性先验”的动机一致。
MSE 的降幅（$23.1\%$）显著大于 SAD（$8.5\%$），可由误差结构解释：MSE 对大偏差呈二次惩罚，
而翻转平均与边缘投影主要消除的恰是边界处少数较大的偏差，对这类离群误差的抑制在 MSE 上被放大体现。
BSP 在已较优的 TTA-GF 上仍带来稳定的小幅改善，且作为保守后处理不引入回退；
其增益虽小，却体现了空间先验与边界先验的正交性——前者管范围、后者管对齐，互不冲突。

\subsection{上游消融与定位质量}

\subsection{应用：RGBA 与 Composer}
\label{sec:exp-app}
将优化后的 $\alpha$ 作为 RGBA 的透明通道即得透明前景，并据 \equref{eq:matting} 完成换背景与虚化。
该环节不重新预测、不计入指标，用于检验 $\alpha$ 接入真实编辑后的边界质量。
$\alpha$ 作为 RGBA 的第四通道，使前景以连续透明度从背景中分离；据 \equref{eq:matting}
将其与新背景（换背景）或原图的模糊版（虚化）合成即可。合成是对 $\alpha$ 的严苛检验：
任何边界偏移都会以白边、黑边或残影的形式被放大。\figref{fig:composer} 显示，
换背景与虚化后主体边缘仍贴合紧密、三种输出轮廓一致，无上述伪影，
表明边界优化不仅反映在指标上、也经得起最终编辑的放大。

\subsection{误差来源分析}
\label{sec:exp-error}
观察残差，本方法的误差主要来自三类情形。其一是\emph{基座的系统性偏差}：
当目标含大面积透明或与背景同色的区域时，ZIM 自身的 $\alpha$ 已显著偏离，
先验只能细化边界、无法纠正这类区域性错误——这也解释了为何边界类指标（Boundary~SAD）的相对改善
远大于整体 SAD。其二是\emph{提示偏移}：当 official bbox 因遮挡或多目标而明显偏移时，
BSP 的“目标必在框内”假设会过度抑制贴近框边的合法前景，框的轻微外扩与边缘模糊只能缓解而非根除。
其三是视频中的\emph{快速运动}：相邻帧对应关系减弱时，时序一致性先验的收益随之下降，
这与\tabref{tab:video} 中各法在 VideoMatt 上普遍弱于 YouTubeMatte 的现象相符。
总体而言，先验擅长修正\emph{边界级}误差，而对\emph{区域级}的基座偏差无能为力——这是零训练范式的固有边界。

\section{讨论与局限}
\label{sec:discussion}
零训练换来了强复现性与模块解耦，但也带来内在边界。性能上界受抠图基座制约：
$\alpha$ 质量本质由 ZIM 决定，先验只能在其输出上无训练地细化，无法弥补基座在极端材质上的系统性偏差。
方法对提示质量敏感：若上游给出明显偏移的 bbox，BSP 的“目标必在框内”假设会失效，
我们以框外扩与边缘模糊缓解但不能根除。此外，TTA 与多尺度带来额外前向开销，在低时延场景需权衡；
视频端 TC 与精度存在折中，双向平滑取了“精度近乎无损”的一端，若更看重稳定可切换更激进的策略。
这些也指明了后续方向：把一致性先验统一为可学习的轻量正则，并在更大规模数据上验证。

\noindent\textbf{何时适用。}本文范式在两类情形下尤为合适。其一，当缺乏抠图标注或算力、难以训练专用模型时，
零训练方案可即取即用、便于复现。其二，当基础模型快速迭代时，解耦设计允许在不改动先验的前提下
替换上游定位器或抠图基座，平滑地享受基座升级带来的红利。反之，若任务对单帧极限精度有严苛要求、
且具备充足标注与算力，端到端微调可能取得更高上界——这与本文定位并不冲突，而是互补的两端：
我们提供的是一个高性价比、可解释且易迁移的起点。

\section{结论}
\label{sec:conclusion}
我们提出零训练框架 PromptMatte，将 SAM2 定位、ZIM 抠图、视频稳定与背景编辑统一为单一推理流水线。
其核心是一条跨域复用的一致性先验：在图像上以 TTA-GF 与 BSP 的形式注入 ZIM 输出，
在视频上推广为双向时序平滑，并由首帧复用把两条分支接成一体。
实验表明本方法在 valid706 上一致优于 ZIM 基线，在视频上以近乎无损的精度提升时序一致性。
